# NixOS Configuration

My NixOS configuration files.

## How to use it?

* Clone this repo in whatever place you find better
* Make a symlink to `/etc/nixos/configuration.nix`
    ```
    $ cd nixos-config
    $ sudo ln -s $PWD/configuration.nix /etc/nixos/configuration.nix
    ```
